How the Web works

The web is a cluster of computers communicating with each other. When a person surfs on the the internet, their computer send a request via HTTP protocol to the server and the browser interprets the HTML documents returned from the server and displayed on the screen.

HTML

HTML stands for Hypertext Markpup Language, forming the majority content of the web.
HTML has two parts: "text" is the content we see on the web and "markup" is how the text is presented.

Tags and Elements
HTML documents are composed of HTML elements. While constructing an HTML document, we use HTML tags to tel the browser the type of each elements. The basic sturcutre of an elements is start tage - content - close tag . However, there are also some tags that do not need a close tag, called the void tag.

There are two types of elements, the inline tag and the block type.
The block tag would make an invisible box surrounding its content, like the ptag.
